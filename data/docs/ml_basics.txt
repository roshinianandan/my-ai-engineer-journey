Machine learning is a branch of artificial intelligence that enables systems to learn from data. There are three main types: supervised learning uses labeled data, unsupervised learning finds hidden patterns, and reinforcement learning trains through rewards. Deep learning is a subset that uses neural networks with many layers. Popular frameworks include TensorFlow, PyTorch, and scikit-learn. Transfer learning allows models trained on one task to be applied to another, saving significant time and compute.

RAG stands for Retrieval Augmented Generation. It combines a retrieval system with a language model. The retrieval system finds relevant documents and the language model generates answers grounded in those documents. This prevents hallucinations because the model only answers from provided context. ChromaDB is commonly used as the vector store in RAG systems.

Embeddings are numerical representations of text. They capture semantic meaning so similar texts have similar vectors. The process of chunking splits long documents into smaller pieces before embedding. Chunk size and overlap affect retrieval quality significantly. Smaller chunks are more precise but may miss context. Larger chunks provide more context but may include irrelevant information.
